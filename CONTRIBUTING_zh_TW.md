# 貢獻於 Activepieces

首先，感謝你撥冗參與貢獻！❤️

我們鼓勵並重視所有類型的貢獻。請參閱 [貢獻指南](https://www.activepieces.com/docs/build-pieces/building-pieces/start-building) 以了解各種參與方式，以及本專案如何處理這些貢獻的細節。在開始貢獻之前，請務必閱讀相關章節，這將使維護者的工作更加輕鬆，並讓所有參與者都有更順暢的體驗。社群期待你的貢獻！🎉

> 如果你喜歡這個專案，但目前沒有時間參與貢獻也沒關係。還有其他簡單的方式可以支持本專案並表達你的肯定，我們也會對此感到非常高興：
> - Star 專案
> - 發 Tweet 分享
> - 在你專案的 README 中引用本專案
> - 在當地的 meetups 中提到本專案，並告訴你的朋友或同事
