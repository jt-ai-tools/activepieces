# 🤯 歡迎來到 Activepieces

這是一款全方位 AI 自動化工具，旨在透過以 **TypeScript** 編寫的 **類型安全 (type-safe)** 組件框架來實現 **可擴展性**。
當您為 Activepieces 貢獻組件時，它們會自動成為 MCP 伺服器，您可以透過 Claude Desktop、Cursor 或 Windsurf 與 LLM 配合使用！

<br>
<br>

## 🔥 為什麼 Activepieces 與眾不同：

- **💖 深受喜愛**：直觀的界面，為技術和非技術用戶提供絕佳體驗，學習曲線短。

<img src="/docs/resources/templates.gif">

- **🌐 開放生態系統：** 所有組件均為開源並可在 npmjs.com 上取得，**60% 的組件由社群貢獻**。

- **🛠️ 最大的開源 MCP 工具包**：我們所有的組件 (280+) 均可作為 MCP 使用，您可以在 Claude Desktop、Cursor 或 Windsurf 上與 LLM 配合使用。

- **🛠️ 組件採用 Typescript 編寫**：組件是 TypeScript 中的 npm 套件，提供完整的自定義功能和最佳的開發體驗，包括在您的電腦上進行 **本地** 組件開發的 **熱重載 (hot reloading)**。 😎

<img src="/docs/resources/create-action.png" alt="">

- **🤖 AI 優先**：原生 AI 組件讓您可以嘗試各種供應商，或使用我們的 AI SDK 建立您自己的代理 (agent)，幫助您在建構器中建立流程。

- **🏢 企業級**：開發人員設置工具，組織中的任何人都可以使用無程式碼建構器。從品牌形象到控制權的完全自定義。

- **🔒 設計安全**：可自託管 (self-hosted) 並支持網絡隔離 (network-gapped)，以實現對數據的最大安全性和控制。

- **🧠 人機協作 (Human in the Loop)**：可以延遲執行一段時間或需要人工審核。這些只是建立在組件框架之上的組件，您可以建立許多類似的組件。 🎨

- **💻 人類輸入界面**：內建支持人類輸入觸發器，如「聊天界面」 💬 和「表單界面」 📝



## 🛠️ 建構器功能：

- [x] 迴圈 (Loops)
- [x] 分支 (Branches)
- [x] 自動重試 (Auto Retries)
- [x] HTTP 請求
- [x] 使用 **NPM** 編寫程式碼
- [x] 在程式碼組件中詢問 AI (非技術用戶可以在不知道如何編碼的情況下清理數據)
- [x] 流程具有完整的版本控制
- [x] 多語言翻譯
- [x] 可自定義模板
- [x] 200+ 組件，請造訪 https://www.activepieces.com/pieces

**我們經常發布更新。請查看產品更新日誌以了解最新功能。**


## 🔌 建立您自己的組件

Activepieces 支持與 Google Sheets、OpenAI、Discord、RSS 以及超過 200 種其他服務的集成。[查看支持集成的完整列表](https://www.activepieces.com/pieces)，由於社群的貢獻，該列表正在不斷擴大。

作為一個**開放生態系統**，所有集成原始碼都可以在我們的存儲庫中找到。這些集成具有版本控制，並在貢獻後直接[發布](https://www.npmjs.com/search?q=%40activepieces)到 npmjs.com。

您可以使用我們的 TypeScript 框架輕鬆建立自己的集成。有關詳細說明，請參閱我們的[貢獻者指南](https://www.activepieces.com/docs/build-pieces/building-pieces/overview)。
