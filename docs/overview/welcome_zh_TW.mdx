# 🔥 為什麼 Activepieces 與眾不同：

- **💖 深受每個人喜愛**：直觀的介面為技術和非技術使用者提供卓越的體驗，學習曲線極短。
  

![](/resources/templates.gif)
- **🌐 開放生態系統**：所有組件皆為開源並可在 npmjs.com 取得，**60% 的組件由社群貢獻**。

- **🛠️ 組件使用 TypeScript 編寫**：組件是 TypeScript 編寫的 npm 套件，提供完整的自定義功能與最佳的開發者體驗，包括在您的機器上進行**本地**組件開發時的**熱重載 (hot reloading)**。 😎


![](/resources/create-action.png)


- **🤖 AI 就緒 (AI-Ready)**：Activepieces 內建原生 AI 組件和代理 (agents)。將 AI 整合到您的流程中既無縫又簡單——嘗試熱門的提供商，或使用我們易於使用的 AI SDK 快速建立自定義代理。

- **🏢 企業級就緒 (Enterprise-Ready)**：開發人員設定工具，組織中的任何人都可以使用無程式碼構建器。從品牌形象到控制權限皆可完整自定義。

- **🔒 安全設計 (Secure by Design)**：可自託管並支援網路隔離 (network-gapped)，確保對您的數據擁有最大的安全性和控制權。

- **🧠 人工介入 (Human in Loop)**：延遲執行一段時間或需要審核。這些只是基於組件框架構建的組件，您可以構建許多類似的組件。 🎨
