### 身份驗證 (Authentication)：

API 使用 "API keys" 來驗證請求。您可以在平台儀表板 (Platform Dashboard) 中查看並管理您的 API keys。

建立 API key 後，您可以將其作為 Bearer token 放入 header 中。

範例：
`Authorization: Bearer {API_KEY}`

### 分頁 (Pagination)

所有端點都使用 seek 分頁。若要對結果進行分頁，您可以提供 `limit` 和 `cursor` 作為查詢參數 (query parameters)。

API 回應將具有以下結構：
```json
{
    "data": [],
    "next": "string",
    "previous": "string"
}
```

- **`data`**：包含請求的結果或數據。

- **`next`**：提供下一組結果的起始 cursor（如果可用）。

- **`previous`**：提供前一組結果的起始 cursor（如果適用）。
