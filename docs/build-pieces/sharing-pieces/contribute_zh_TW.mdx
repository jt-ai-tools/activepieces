---
title: "貢獻"
description: "瞭解如何向主儲存庫貢獻組件 (piece)。"
---

<Steps>
   <Step title="提交拉取請求 (Pull Request)">
     - 構建並測試您的組件。
     - 從您的儲存庫向主分叉 (main fork) 提交拉取請求。
     - 維護者將會仔細審查您的工作。
   </Step>
   <Step title="合併拉取請求">
   - 一旦拉取請求獲得批准，它將被合併到主分支 (main branch)。
   - 您的組件將在幾分鐘內可用。
   - 自動化的 GitHub Action 將對其進行打包，並在 npmjs.com 上建立一個 npm 軟體包。
   </Step>
</Steps>
