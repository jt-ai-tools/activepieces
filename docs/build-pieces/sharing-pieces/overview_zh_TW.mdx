## 方式
- [回饋貢獻](/build-pieces/sharing-pieces/contribute_zh_TW): 透過將您的 piece 回饋到主儲存庫來發布您的 piece。
- [社群](/build-pieces/sharing-pieces/community_zh_TW): 直接在 npm 上發布您的 piece 並與社群分享。
- [私人](/build-pieces/sharing-pieces/private_zh_TW): 在 activepieces 上私下發布您的 piece。
