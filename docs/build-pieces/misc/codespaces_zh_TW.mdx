GitHub Codespaces 是一個雲端開發平台，讓開發者可以直接在瀏覽器中撰寫、執行和偵錯程式碼，並與 GitHub 無縫整合。

### 設定 Codespaces 的步驟

1. 前往 [Activepieces repo](https://github.com/activepieces/activepieces)。

2. 點擊 Code `<>`，接著在 codespaces 下點擊 create codespace on main。

![Create Codespace](/resources/screenshots/development-setup_codespaces.png)

<Note>
  預設情況下，開發設定僅會建置特定的 piece。請開啟 `packages/server/api/.env` 檔案並新增以逗號分隔的 piece 列表以供使用。

更多詳細資訊，請參閱 [Piece Development](/build-pieces/building-pieces/development-setup_zh_TW#pieces-development) 章節。

</Note>

3. 開啟終端機並執行 `npm start`

4. 開啟 port 4200 以存取前端網址，並使用以下詳細資訊登入：

Email：`dev@ap.com`
Password：`12345678`
