您可以監控由已啟用的流程所產生的每一次執行：

1. 前往儀表板，點擊 **Runs**。
2. 找到您要查看的執行記錄，然後點擊它。
3. 您將在唯讀模式下看到編排器，每個步驟都會顯示 ✅ 或 ❌ 以表示其執行狀態。
4. 點擊任何一個步驟，您將在 **Run Details** 面板中看到 **input** 和 **output**。
