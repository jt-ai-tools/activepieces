title: "版本歷史"
icon: 'clock'
description: "了解 Activepieces 中的流程版本控制運作方式"
---

Activepieces 會追蹤所有已發佈的流程及其版本。以下是其運作方式：

1. 您可以在 **草稿 (draft)** 模式下隨意編輯流程。
2. 完成變更後，您可以將其發佈。
3. 已發佈的版本將被鎖定且無法編輯。
4. 如果您嘗試編輯已發佈的流程，若目前沒有草稿，Activepieces 會建立一個新的 **草稿 (draft)**，並將 **已發佈 (published)** 版本複製到新版本中。

這意味著您隨時可以回到先前的版本，並在草稿模式下編輯流程，而不會影響已發佈的版本。
