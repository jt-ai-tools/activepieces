Activepieces 中的專案是您在組織內管理自動化和資源的主要單位。每個專案都擁有獨立的流程 (flows)、連線 (connections) 和表格 (tables)。所有擁有該專案存取權限的成員都能共同使用這些資源。

專案分為兩種類型：
- **個人專案 (Personal Projects)**：每位受邀加入組織的使用者都會自動獲得一個個人專案。這是一個私有空間，僅限該使用者建立及管理流程、連線和表格。
- **團隊專案 (Team Projects)**：團隊專案是共用空間，可在此頁面進行建立與管理。您可以邀請多位使用者加入團隊專案，以便協作、共享流程、連線及表格的存取權限並共同開展工作。

在組織您的工作時，請針對團隊協作建立團隊專案，並將個人專案用於個人或私密任務。
