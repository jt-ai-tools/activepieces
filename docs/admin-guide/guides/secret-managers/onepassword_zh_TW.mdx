title: "1Password"
description: "將 1Password 連接到 Activepieces 以進行集中式金鑰管理"

1Password Secrets Automation 讓您可以安全地儲存並從 1Password 保險箱 (vaults) 中擷取憑證。此整合使用 **服務帳戶 Token (service account token)** 透過 1Password SDK 進行身分驗證。

## 前置作業

- 具有 **Teams** 或 **Business** 方案的 1Password 帳戶（服務帳戶需要付費方案）
- 在您的 1Password 帳戶中具有建立服務帳戶的權限

## 步驟 1 — 建立服務帳戶

1. 登入 [1password.com](https://1password.com) 並前往 **Developer Tools → Service Accounts**。
2. 點擊 **Create a service account**。
3. 為其命名（例如 `Activepieces`）。
4. 針對 Activepieces 需要存取的保險箱授予 **Read Items** 權限。
5. 點擊 **Create service account** 並複製 Token — 它以 `ops_` 開頭且僅顯示一次。

<Warning>
  請安全地儲存 Token。一旦您離開此頁面，1Password 將不會再次顯示該 Token。
</Warning>

## 步驟 2 — 在 Activepieces 中連接

1. 前往 **Platform Admin → Security → Secret Managers**。
2. 從供應商列表中選擇 **1Password**。
3. 貼上步驟 1 中的 **Service Account Token**。
4. 點擊 **Connect** 以測試並儲存連線。

## 在連線中使用 1Password

設定需要憑證的全域連線時：

1. 點擊憑證欄位旁邊的 **鑰匙圖示** (🔑)。
2. 選擇 **1Password** 作為金鑰管理員 (secret manager)。
3. 輸入以下格式的 **Secret Reference**：
   ```
   op://vault/item/field
   ```
   - **vault** — 保險箱名稱或 ID（例如 `Production`）
   - **item** — 項目標題或 ID（例如 `Stripe`）
   - **field** — 項目內的欄位標籤（例如 `password`、`api key`）

Activepieces 將從 1Password 擷取金鑰值，並在執行時將其注入連線中。

<Tip>
  獲取精確引用最簡單的方法是開啟 1Password 桌面應用程式，右鍵點擊任何欄位，然後選擇 **Copy Secret Reference**。這將複製準備好貼上的 `op://...` 字串。
</Tip>

## 金鑰引用範例

| 1Password 位置 | 引用 (Reference) |
|---|---|
| 保險箱 `Production` → 項目 `Stripe` → 欄位 `Secret Key` | `op://Production/Stripe/Secret Key` |
| 保險箱 `Shared` → 項目 `Database` → 欄位 `password` | `op://Shared/Database/password` |

<Warning>
  如果您更新了現有的金鑰但看不到反映的更新，請參考 [快取](/admin-guide/guides/secret-managers/overview_zh_TW#caching)。
</Warning>
