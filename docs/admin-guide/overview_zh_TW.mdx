**平台管理 (Platform Admin)** 是管理您的 Activepieces 執行個體的集中式管理面板。它專為希望全面控制使用者、整合、安全性及內部自動化的團隊和組織而設計。

## 您可以做什麼？

透過平台管理，您可以：

- **自訂品牌 (Custom Branding)：** 調整 Activepieces 的外觀以符合您組織的形象，包括顏色、標誌和字體。

- **專案管理 (Project Management)：** 為內部團隊和使用者建立、編輯和組織專案。

- **組件管理 (Piece Management)：** 控制可用的整合組件 (Pieces)，包括管理用於團隊工作流程的自訂或內部組件。

- **使用者管理 (User Management)：** 新增和移除使用者、傳送邀請，並分配角色和權限。

- **AI 供應商管理 (AI Provider Management)：** 設定並管理可在您的流程中使用的 AI 供應商（如 OpenAI、Anthropic 等）。

- **SSO 與安全性 (SSO & Security)：** 設定單一登入 (SSO) 供應商並管理安全性設定，以確保您的執行個體安全。
