## 調色盤

![調色盤](/resources/color-palette.png)

此調色盤包含：

- **Primary colors（主色）**：用於主要操作和品牌識別
- **Secondary colors（輔助色）**：用於輔助元素
- **Semantic colors（語意化顏色）**：用於狀態和回饋（success、warning、destructive）


## 技術棧

我們的前端架構包含：

- **React** - 核心 UI 框架
- **Shadcn UI** - 元件庫 
- **Tailwind CSS** - Utility-first（效用優先）樣式
