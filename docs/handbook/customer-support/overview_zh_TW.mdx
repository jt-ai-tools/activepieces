在 Activepieces，我們對客戶支援採取了獨特的方式。我們沒有專門的支援人員，而是由全職工程師輪流處理支援請求。這確保了您能從產品構建者那裡獲得專業的技術幫助。

### 支援排班

我們的值班工程師將值班期間的一部分時間用於處理客戶支援。欲了解更多關於其運作方式的詳情，請查看我們的值班文件。


### 支援管道

- 社群支援 
    - GitHub Issues：我們積極監控並回應 [GitHub 儲存庫](https://github.com/activepieces/activepieces) 上的問題
    - 社群論壇：我們在 [社群平台](https://community.activepieces.com/) 上與用戶互動，提供幫助並收集回饋
    - 電子郵件：僅限於帳戶相關問題、刪除帳戶請求或帳單問題。

- 企業支援
    - 企業客戶透過 Slack 獲得專屬支援
    - 我們使用 [Pylon](https://usepylon.com) 來高效管理支援工單和客戶頻道
    - 有關使用 Pylon 的詳細資訊，請參閱我們的 [Pylon 指南](/handbook/customer-support/pylon_zh_TW)
