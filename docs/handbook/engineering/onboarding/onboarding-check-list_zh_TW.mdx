🎉 歡迎來到 Activepieces！

本指南為新進員工入職流程提供了查核清單。

---

## 📧 基礎事項

- [ ] 設定您的 @docs/resources/screenshots/cloud.activepieces.com_projects_er7zUIumSRjbOuqubZiye_flows_limit=10(laptop) (1).png:Zone.Identifier 電子郵件帳戶並設定 2FA
- [ ] 確認已加入我們的私人 Discord 伺服器。
- [ ] 獲邀加入 Activepieces GitHub 組織並設定 2FA
- [ ] 被指派一名入職夥伴 (buddy)，他將成為您的入職導師。


<Tip>
在您入職的前兩個月，我們將每兩週安排一次 1:1 會議，以確保您的進展順利，並保持雙向溝通。
兩個月後，我們將把 1:1 會議的頻率降至每月一次。
</Tip>


<Tip>
如果您沒有設定 2FA，我們將從安全角度收到警示。
</Tip>

---


### 工程查核清單

- [ ] 使用我們的設定指南設定您的開發環境
- [ ] 了解儲存庫結構和我們的技術棧 (Fastify, React, PostgreSQL, SQLite, Redis)
- [ ] 了解關鍵的資料庫表 (Platform, Projects, Flows, Connections, Users)
- [ ] 在入職第一天內完成您的第一個「熱身 (warmup)」任務 (這是我們的傳統！)

## 🌟 成功祕訣

- 不要猶豫提問——團隊在您入職的最初幾天會特別熱心提供協助
- 花時間從業務角度了解產品
- 與您的入職夥伴緊密合作，快速上手
- 查閱我們的文件、探索程式碼庫，並查看您負責範圍之外的社群資源。
- 定期提供您的想法和回饋
