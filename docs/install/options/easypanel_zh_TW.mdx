Easypanel 是一款現代化的伺服器控制面板。如果您在伺服器上 [運行 Easypanel](https://easypanel.io/docs)，您可以透過它一鍵部署 Activepieces。

<a target="_blank" rel="noopener" href="https://easypanel.io/docs/templates/activepieces">![部署到 Easypanel](https://easypanel.io/img/deploy-on-easypanel-40.svg)</a>

## 操作步驟

1. 在您的雲端服務供應商上建立一個運行 Ubuntu 的虛擬機器 (VM)。
2. 按照官方網站的說明安裝 Easypanel。
3. 建立一個新專案。
4. 使用專用模板安裝 Activepieces。
