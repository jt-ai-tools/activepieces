# 為什麼 Activepieces 需要資料？

作為一個自我託管（self-hosted）產品，收集使用指標和洞察對我們來說可能很困難。然而，這些分析對於協助我們理解關鍵行為並提供符合您需求的高品質體驗至關重要。

為了確保我們能持續改進產品，我們決定追蹤某些對於理解 Activepieces 使用情況至關重要的基本行為和指標。

我們實施了最小化追蹤計畫，並在獨立章節中提供了收集指標的詳細列表。
