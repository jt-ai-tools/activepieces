---
title: "重設密碼"
description: "如何在自行代管（self-hosted）的實例上重設密碼"
icon: "key"
---

如果您在自行代管的實例上忘記了密碼，可以按照以下步驟進行重設：

1. **尋找 PostgreSQL Docker 容器**：
   - 使用 `docker ps` 等命令來尋找 PostgreSQL 容器。

2. **進入容器**：
   - 使用以下命令進入 PostgreSQL Docker 容器。
   ```bash
   docker exec -it POSTGRES_CONTAINER_ID /bin/bash
   ```

3. **開啟 PostgreSQL 控制台**：
   - 在容器內，使用 `psql` 命令開啟 PostgreSQL 控制台。
   ```bash
   psql -U postgres
   ```

4. **連接到 ActivePieces 資料庫**：
   - 連接到 ActivePieces 資料庫。
   ```sql
   \c activepieces
   ```

5. **建立安全的密碼**：
   - 使用像 [bcrypt-generator.com](https://bcrypt-generator.com/) 這樣的工具生成一個新的安全密碼，疊代次數（number of rounds）請設定為 10。

6. **更新您的密碼**：
   - 在 PostgreSQL 控制台中執行以下 SQL 查詢，將 `HASH_PASSWORD` 替換為您產生的新密碼雜湊值，並將 `YOUR_EMAIL_ADDRESS` 替換為您的電子郵件。
   ```sql
   UPDATE public.user_identity SET password='HASH_PASSWORD' WHERE email='YOUR_EMAIL_ADDRESS';
   ```
