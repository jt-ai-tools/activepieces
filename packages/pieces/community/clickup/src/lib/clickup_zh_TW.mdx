## 如何取得清單 (List) ID

要取得清單 ID：

1. 前往 ClickUp 應用程式 (https://app.clickup.com/)，選擇一個空間 (Space) 並進入該空間內的某個清單
2. 網址列中的網址看起來應該像這樣：https://app.clickup.com/2499706/v/l/li/901000754585
3. 複製最後一個斜線 (/) 後方的數字即可

## 如何取得空間 (Space) ID

要取得空間 ID：

1. 前往 ClickUp 應用程式 (https://app.clickup.com/) 並選擇進入一個空間
2. 網址列中的網址看起來應該像這樣：https://app.clickup.com/2499706/v/l/s/38651852
3. 複製最後一個斜線 (/) 後方的數字即可

## 如何取得團隊 (Team) ID

要取得團隊 ID：

1. 前往 ClickUp 應用程式 (https://app.clickup.com/)
2. 選擇您偏好的工作區 (Workspace)（如果尚未選擇）
3. 網址列中的網址看起來應該像這樣：https://app.clickup.com/2499706/home
4. 複製緊接在 app.clickup.com/ 後方的數字 (https://app.clickup.com/[在此複製數字]/home)
