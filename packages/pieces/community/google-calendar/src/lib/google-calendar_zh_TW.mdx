## 設定並執行呼叫 Google 日曆 (Calendar) API 的應用程式。

1. 在 Google Cloud 控制台中，啟用 Google 日曆 API。
2. 在 Google Cloud 控制台中，前往「選單」 > 「API 與服務」 > 「憑證」。
3. 點擊「建立憑證」 > 「OAuth 用戶端 ID」。
4. 在「名稱」欄位中，輸入此憑證的名稱。此名稱僅會顯示在 Google Cloud 控制台中。
5. 點擊「建立」。出現「已建立 OAuth 用戶端」畫面，顯示您的新「用戶端 ID」和「用戶端密鑰」。
6. 點擊「確定」。新建立的憑證將顯示在「OAuth 2.0 用戶端 ID」下方。
