## 取得傳送訊息的 Webhook 網址 (Webhook URL)

1. 前往您想要新增 Webhook 的 Discord 伺服器
2. 點擊伺服器設定圖示（外觀像個齒輪）
3. 向下捲動到「整合」(Integrations) 部分，並點擊「Webhooks」
4. 為您的 Webhook 填入名稱和頭像，並選擇您想要讓 Webhook 貼文的頻道
5. 點擊「建立」(Create) 按鈕
6. **複製產生的 Webhook 網址**。這就是您之後透過機器人傳送訊息到 Discord 頻道時會用到的網址
7. **請務必將 Webhook 網址保密，僅分享給信任的對象**
8. 取得 Webhook 網址後，您就可以在 Activepieces 中使用它

## 選項

使用者名稱 (選填): Webhook 的顯示名稱
內容: 要傳送的訊息內容
頭像網址 (選填): Webhook 的頭像網址
嵌入內容 (選填): 隨 Webhook 訊息傳送的嵌入內容 (Embeds)，請參考[這裡](https://birdie0.github.io/discord-webhooks-guide/structure/username.html)
TTS (選填): 使用機器人語音大聲朗讀訊息
